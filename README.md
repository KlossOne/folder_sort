# folder_sort
Organisation automatique à partir d'un dossier

# But
Nous nous somme aperçu que certains dossier devenais bordélique avec le temps (Ex : Download/), et avons décider de crée un tools pour le trier automatiquement.

# Ex :
Ce dossier : 
```
Download/
├── film1.avi
├── musique1.mp3
├── Serie1.ep1.avi
├── Serie1.ep2.avi
├── Serie2.ep1.avi
```

deviendrais : 

```
Download/
├── Film/
│   └── film1.avi
├── Musique/
│   └── musique1.mp3
└── Serie/
    ├── Serie1/
    │   ├── Serie1.ep1.avi
    │   └── Serie1.ep2.avi
    └── Serie2/
        └── Serie2.ep1.avi
```
